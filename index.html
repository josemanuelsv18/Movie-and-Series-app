Recommertainment is a web app that takes public APIs from diferents databases and web pages to give our users the best recommendation in movies, series and anime. You can easily access to the most popular or best rated movies and series in a period of time or at all; see where to watch it and extra info about it.

For Movies and series we use the API from The Movie Database, a collaborative database about movies and series. all info about the movies is provided by this site, all this web page do is organize it to give recommendations to our users using TMBD data
